Attestation covid 2020

Comment utiliser le site:

1. Creer une url avec les params suivants: ils sont tous obligatoires

  • prenom: String
  • nom: String
  • dateNaissance: String, 10 chars en tout obligatoirement
  • lieuNaissance: String
  • adresse: String,
  • ville: String,
  • codePostal: Int, uniquement 5 chiffres, fuck la corse
  • minutesDepart: Int, le nombre de minutes dans le passe ou a ete remplie l'attestation. Par ex, 15 fera que l'attestation aura une date de remplissage de l'heure courante - 15 minutes.
  • motifs: String, une concatenation separee par des virgules des motifs possibles: ACHATS, TRAVAIL, SANTE, MOTIF_FAMILIAL, SPORT_ANIMAUX, CONVOCATION, MISSIONS, ENFANTS, HANDICAP

Exemple: https://attestationcov.herokuapp.com?prenom=Elric&nom=Melnibone&dateNaissance=01/01/1922&lieuNaissance=dans les etoiles&adresse=34 rue des jonquilles&ville=Paris&codePostal=12345&minutesDepart=12&motifs=TRAVAIL,ENFANTS

2. Mettre cette url en bookmark sur un navigateur, ou ajouter un raccourci depuis le dashboard de votre mobile

L'idee est de creer un bookmark par motif, maniere de pas remplir l'attestation a chaque fois

Bref, vous vous debrouillez hein. Ca devrait telecharger l'attestation.
